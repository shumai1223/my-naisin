# 都道府県別 内申点制度格差の統計的類型化分析（2026年度版）

47都道府県の内申点計算方式（`src/lib/prefectures.ts`）を対象に、①学年重み付けパターン、②実技教科
傾斜配点比率の2軸で統計的に類型化した分析です。卒業論文・学術ブログ等での引用を想定し、算出方法・
元データを明示した形でまとめています。

[[fable5-fullaccel-backlog-2026-07]] TIER X - X-26。X-2（内申点の日本地図）・W-2（内申格差レポート・
`/naishin-kakusa`）のデータをさらに深掘りした詳細版です。

## データソースと算出方法

- 元データ: `src/lib/prefectures.ts`（各都道府県教育委員会の入学者選抜実施要項に基づく `gradeMultipliers`
  ／`coreMultiplier`／`practicalMultiplier` フィールド、`lastVerified` 日付で確認日を記録）。
- 集計方法: `grep`で該当フィールドを機械的に抽出し、47件を目視で県コードと突き合わせて集計（手転記による
  誤記混入を避けるため、抽出結果をソースコードの行番号と対応付けて検証済み）。
- 「実技傾斜比率」は `practicalMultiplier ÷ coreMultiplier` として算出（例: 鹿児島県は
  `practicalMultiplier: 20, coreMultiplier: 2` → 傾斜比率10倍）。

## 分析①: 学年重み付けの3類型

内申点の対象学年（中1〜中3のどの学年をどれだけ重視するか）で47都道府県を分類すると、明確に3つの
類型に分かれます。

| 類型 | 定義 | 該当県数 | 該当県 |
|---|---|---|---|
| A. 中3のみ型 | 中1・中2の重みが0（`gradeMultipliers[1]=0, [2]=0`） | **11県** | 山形・東京・福井・長野・静岡・愛知・三重・兵庫・鳥取・福岡・鹿児島 |
| B. 均等評価型 | 中1〜中3をすべて同じ重みで評価（`1:1:1`） | **23県** | 青森・宮城・秋田・福島・茨城・栃木・群馬・千葉・新潟・石川・山梨・滋賀・京都・岡山・山口・徳島・香川・愛媛・高知・佐賀・長崎・宮崎・沖縄 |
| C. 傾斜型（中間） | 学年ごとに異なる重み（中3を重視する等） | **13県** | 北海道・岩手・埼玉・神奈川・富山・岐阜・大阪・奈良・和歌山・島根・広島・熊本・大分 |

**含意**: A類型（11県）に住む生徒は、理論上「中1・中2の成績が入試の合否に直接影響しない」ため、中3から
の巻き返しが制度上可能です。一方B類型（23県・全体の約49%）では中1からの積み重ねがそのまま入試の内申点
になり、「中1だから関係ない」という認識は誤りとなります。全国の過半数の都道府県がB類型に属することは、
「内申点対策は中3から」という一般に流布しがちな認識と実態がズレている可能性を示唆します。

## 分析②: 実技教科傾斜配点比率の分布

実技4教科（音楽・美術・保健体育・技術家庭）を主要5教科と比べてどれだけ重く扱うかを「傾斜比率」
（実技倍率÷主要教科倍率）として算出し、分布を整理しました。

| 傾斜比率 | 該当県数 | 該当県 |
|---|---|---|
| 1.0倍（傾斜なし） | **31県** | 北海道・青森・山形・茨城・栃木・群馬・埼玉・千葉・神奈川・新潟・富山・石川・福井・長野・岐阜・静岡・愛知・三重・滋賀・大阪・奈良・和歌山・島根・広島・山口・愛媛・福岡・佐賀・長崎・熊本・宮崎 |
| 1.5倍 | 3県 | 岩手・山梨・沖縄 |
| 1.875倍 | 1県 | 兵庫 |
| 2.0倍 | 11県 | 宮城・秋田・福島・東京・京都・鳥取・岡山・徳島・香川・高知・大分 |
| **10.0倍（外れ値）** | **1県** | **鹿児島** |

**含意**: 全体の約66%（31/47県）は実技教科への傾斜が一切なく、主要教科と同倍率で扱われます。傾斜が
ある16県の大半（15県）は1.5〜2.0倍の範囲に収まる一方、**鹿児島県のみ10倍という統計的な外れ値**を示して
います。鹿児島県の内申点の実に9割前後が実技4教科の出来で決まる計算になり、単純な平均・中央値による
比較では代表性を欠く典型例として、統計処理の際は個別に注記する必要があります。

## 分析③: 2軸クロス集計

学年重み付け類型（A/B/C）×実技傾斜の有無（0.5〜2.0倍を「軽度傾斜あり」、10倍の鹿児島を「極端な傾斜」
として区分）でクロス集計すると、以下の傾向が見られます。

- A類型（中3のみ）×傾斜なし: 山形・福井・長野・静岡・三重の5県。「学年重視の絞り込み」と「教科間の
  平等」を両立させる、最もシンプルな制度設計です。
- A類型×傾斜あり: 東京（2倍）・兵庫（1.875倍）・鳥取（2倍）・福岡（傾斜なし・注:福岡はA類型かつ傾斜なし
  のため上の分類に含む）・鹿児島（10倍）・愛知（傾斜なし）。中3のみに絞りつつ実技を重視する設計は、
  短期間で実技科目の巻き返しを図る必要がある点で受験生への負荷が高い構造と言えます。
- B類型（均等評価）×傾斜あり: 京都（2倍）・岡山（2倍）・徳島（2倍）・香川（2倍）・高知（2倍）。中国・
  四国・近畿の一部に、均等評価かつ実技重視という組み合わせが偏在する傾向が見られます（西日本での実技
  重視傾向は今後n数を増やしてさらに検証する余地があります）。

## 学術利用にあたっての注意

- 本分析は都道府県教育委員会が公表する令和8年度（2026年度）入試の実施要項・要綱に基づく一次データを
  機械集計したものです（`prefectures.ts`の`sourceUrl`/`sourceTitle`/`lastVerified`フィールドに全47県分の
  出典を記録）。
- 制度は年度改定される場合があります（例: 奈良県は令和8年度から計算方法を変更）。引用の際は
  `lastVerified`の日付を確認し、最新の公式要項と照合してください。
- 実技傾斜「1.0倍（傾斜なし）」は「実技教科が軽視される」ことを意味しません。主要教科と同倍率で
  扱われる、という意味です。
- 引用・転載時は出典として「My Naishin（https://my-naishin.com）内申点白書2026」の明記にご協力ください。
  関連資料: [/report/2026](https://my-naishin.com/report/2026)（内申点白書2026本体）、
  [/naishin-kakusa](https://my-naishin.com/naishin-kakusa)（内申点格差レポート）。
