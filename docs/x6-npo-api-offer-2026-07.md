## 設計思想・教訓

- 文体はX-2（地方紙）・X-5（行政）・X-7（個人サイト）のいずれとも異なる**「支援活動への敬意→無償提供の
  申し出→リンクは"必須条件でなく感謝の形で"という控えめな位置づけ」**の3段構成で統一。バックログの想定
  通り、営利目的の相互リンクより「支援への感謝」の文脈でリンクしやすい非営利枠の構造を意識した。
- 各団体の活動規模・対象地域（全国/埼玉県内/名古屋市内）に応じて、提案する県の内申点制度の特徴（埼玉=
  学年比率が高校ごとに分岐・愛知=評価方法Ⅰ〜Ⅴ選択制）を差し込み、汎用テンプレートでなく個別性のある
  提案にした。
- 「無償API提供」だが、実際のAPI利用（認証キー発行等）は既存のapi-tiers/api-keys課金ゲート
  （[[session-2026-06-27-api-monetization-eeatB]]）とは別枠の無料枠を新設する必要があり、実際にNPOから
  反応があった場合は👤と相談のうえで提供方法（埋め込みウィジェット/簡易APIキー等）を決める必要がある。
