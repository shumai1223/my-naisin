# 内申点タイプ診断（/naishin-type-shindan）撤退基準

ZZ-5c（診断プロダクト1本）の一部。DoD「撤退基準を先に定義」に対応する文書。
撤退判断はGA4 MCP実測（週次成長ログのサイクル）で行い、この文書に照らして機械的に判定する。

## 計測する指標

- **分子**: `share_to_parent`（source='naishin-type-shindan'でフィルタ・ZZ-5bのmedium別内訳込み）
- **分母**: `save_result_cta_view`（source='naishin-type-shindan'でフィルタ・ZZ-2d実装）
- **共有率** = 分子 / 分母

補助指標（撤退判断の参考・単独では撤退理由にしない）:
- `naishin_type_result`（type_id別の分布・診断自体がどれだけ完了しているか）
- `lead_submit`（source='naishin-type-shindan'・名簿化への貢献度）

## 撤退基準

**共有率が4週連続で3%を下回ったら撤収する。**

- 閾値3%の根拠: 本機能公開時点でサイト全体の共有率の実測ベースラインが存在しないため、
  暫定値として設定した（他の診断系導線の一般的な目安として仮置き）。
  **実測データが十分に蓄積した最初のjudge（4週目）で、サイト内の他の共有導線
  （SaveResultCTA全体・偏差値診断等）の実測共有率と比較し、この閾値が妥当かを見直すこと。**
- 「4週連続」とする理由: 単発の低迷（学校行事・長期休暇等の季節要因）による誤判定を避けるため。
- 分母（save_result_cta_view）が週20件未満の週は、n不足として判定week数にカウントしない
  （母数が薄い状態での偶然の低下を「連続未達」に含めない）。

## 撤退した場合の対応

- ページ自体は削除しない（既存の内部リンク・被リンクを死なせないため）。
- `/tools`一覧・関連ツール導線からのリンクを外し、新規流入の入口を閉じる。
- 既存のURLはリダイレクトせずそのまま残す（訪問者には引き続き診断結果を提供する・撤退は「新規の宣伝・導線を止める」の意味であり「機能を壊す」ではない）。
- 撤退の判断・実施はloop-question-noteに記録し、👤の確認を経てから実施する（新規ページの取り下げに類する判断のため）。

## 継続する場合

4週間の共有率が閾値以上、または分母が n<20 のまま週次で成長を続けている場合は継続する。
継続判定・共有率の実測値は`docs/weekly-growth-log.md`の該当週エントリに記録する。
