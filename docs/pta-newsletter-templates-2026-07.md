[[fable5-fullaccel-backlog-2026-07]] TIER X - X-18。地域のPTAブログ・保護者会の学年だより・学級通信
から参照・転載されることを狙い、「そのままPTA便りに転載できる分量・文体」の内申点解説テンプレートを
作成する。W-5（寄稿ドラフト・メディア向け・約2,000字）とは異なり、こちらは**PTA便り1コーナー分

## テンプレート①「三者面談前に知っておきたい内申点の基礎」

**想定用途**: 2学期の三者面談前（10〜11月頃）に配布される学年だより・学級通信のコラム欄

```
━━━━━━━━━━━━━━━━━━━━━━
三者面談前に知っておきたい「内申点」の基礎
━━━━━━━━━━━━━━━━━━━━━━

もうすぐ三者面談。「内申点」という言葉が飛び交いますが、実はこの言葉、都道府県ごとに
中身が違うことをご存知でしょうか。

内申点とは、中学校の通知表の評定（5段階評価）をもとに、高校入試の合否判定に使えるよう
点数化したものです。ただし、

・どの学年の成績が対象になるか（中1〜中3全部？　中3だけ？）
・音楽・美術・保健体育・技術家庭の「実技教科」がどれだけ重視されるか
・満点が何点で、学力検査とどんな比率で合算されるか

は都道府県によって大きく異なります。三者面談で先生から「内申点は〇〇点」と言われても、
それが何点満点で、当日の学力検査とどんな比率で合算されるのかまで確認しておくと、
残りの学期をどう過ごすべきかがより具体的に見えてきます。

お子さんの都道府県の内申点の仕組みを詳しく知りたい方は、内申点計算サイト「My Naishin」
（https://my-naishin.com）で無料で確認できます。47都道府県すべての教育委員会の公式
発表をもとにしたデータです。

──────────────────
情報提供：内申点・偏差値計算サイト「My Naishin」（https://my-naishin.com）
※本コーナーの転載・引用は自由です。転載の際は上記クレジットの記載にご協力ください。
──────────────────
```

## テンプレート②「うちの子の内申点、実は県によって重みが違う」

**想定用途**: 転校生を多く受け入れる地域や、県境の学校のPTA便り・教育委員会だより

```
━━━━━━━━━━━━━━━━━━━━━━
うちの子の内申点、実は県によって重みが違う？
━━━━━━━━━━━━━━━━━━━━━━

「オール4」「オール3」といった内申点の話をする際、実はその「重み」が都道府県によって
大きく異なることをご存知でしょうか。

例えば、実技4教科（音楽・美術・保健体育・技術家庭）を主要5教科よりどれだけ重く扱うかは
県ごとにバラバラです。実技教科への傾斜（重み付け）がまったくない県もあれば、実技教科の
出来が総合得点の大部分を左右する県もあります。

また、内申点にどの学年の成績を使うかも県によって違います。中1〜中3すべての成績が
対象になる県もあれば、中3の成績だけが対象になる県もあります。転勤・引っ越しなどで
県をまたぐ可能性があるご家庭は、この違いを知っておくと安心です。

47都道府県すべての違いを比較できるページを、内申点計算サイト「My Naishin」
（https://my-naishin.com/naishin-kakusa）で公開しています。すべて各都道府県教育
委員会の公式発表にもとづくデータです。

──────────────────
情報提供：内申点・偏差値計算サイト「My Naishin」（https://my-naishin.com）
※本コーナーの転載・引用は自由です。転載の際は上記クレジットの記載にご協力ください。
──────────────────
```

## 設計思想・教訓

- W-5（メディア向け寄稿ドラフト・約2,000字・実名メディアへの個別売り込み前提）とは狙いを分けている。
  こちらは**特定の送付先を決めず、汎用テンプレートとして先に用意しておく**方式（「クレジット表記を
  残していただければ自由に転載可」という緩い条件のため、個別の許諾交渉が不要）。
- 分量はPTA便り・学級通信の1コーナー分を想定し400〜600字に抑えた（W-5の約2,000字より大幅に短い）。
  専門用語（K値・S値等）は使わず、保護者が読んですぐ理解できる語彙のみを使用。
- 内容はすべて既存の検証済みページ（/naishin-kakusa・prefectures.ts）のデータの要約であり、新規の
  事実確認・数値の断定は一切していない。
- クレジット表記（情報提供：My Naishin＋URL）を末尾に固定フォーマットで入れることで、印刷物に
  転載された場合でも出典・被リンク元が明示される設計（五ツ木書房パターンと同様の「使ってもらうことで
  被リンクが発生する」設計）。
