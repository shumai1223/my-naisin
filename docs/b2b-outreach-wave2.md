# B2B送信用完成メール集 wave2（模試会社セグメント・2026-07-15版）

[docs/b2b-outreach-ready-2026-07.md](./b2b-outreach-ready-2026-07.md)（wave1・塾管理SaaS/教育メディア20社・送信済み）の第2波。
対象は**都道府県別の公開模試を運営する会社**に絞った。内申点×合否判定はまさに模試会社の本業であり、
「47都道府県の内申計算をAPIで」が刺さりやすいセグメントという想定（[[fable5-fullaccel-backlog-2026-07]] V-3）。

**実在確認の方法**: WebSearchで各社の公式サイト・会社概要・お問い合わせ窓口を個別に確認（2026-07-15実施）。
wave1の教訓（Comiruが「塾・スクール名」必須の顧客専用フォームだった前科）を踏まえ、
「実力テスト会に関するお問い合わせ（**その他の方**）」のように一般法人向けと明示された窓口を優先した。

**送信前チェック**（wave1と同じ）: ①宛先URLを開いて現存を確認 ②署名は「しゅうまい／My Naishin運営／naishin.dev@gmail.com」で統一
③フォームに「塾名」「契約状況」等の必須項目がないか目視確認してから送信。

**CTA設計はwave1を踏襲**（Pro自己申込みが主導線・打ち合わせは任意）が、模試会社向けには
**掲載枠（スポンサー直販）**の1行を新規同梱した——模試の結果帳票・Webマイページに「内申点計算はMy Naishin」の
形でスポンサー掲載する広告的な提案も選べるようにするため。

**⚠️今回は8社で確定・打ち止め（20社中）**: WebSearchで実在確認した候補のうち、以下は載せなかった。
- **新教育産業「Wもぎ」**: 運営元（吉野創育/新教育研究協会）の特定に成功したが、検索結果に「Wもぎは2024年度開催休止です」という情報が混在し、現在も稼働中か確証が持てなかったため除外。
- **さなる（佐鳴予備校）・アップ教育企画**: 学習塾チェーン本体であり、独立した都道府県公開模試ブランドを外部に展開している一次情報を確認できなかったため「模試会社」セグメントとしては不適合と判断し除外。
残り12社は次回ループ（V-3続き）でWebSearchを継続する。

## 次回ループへの引き継ぎ（残り候補・未検証）

以下は都道府県別模試の存在は一般に知られているが、2026-07-15時点でまだWebSearch未実施、または
実在確認が不十分（会社名の表記ゆれ・運営会社の特定に至らず）。次回V-3続きで検証すること。

- 静岡県：中学生対象の公開模試運営会社（さなるは学習塾チェーン本体のため除外済み・別の模試専業社を探す）
- 広島県：県内公開模試の運営会社（アップ教育企画は学習塾チェーンのため除外済み）
- 岡山県・香川県・愛媛県など四国・中国地方の公開模試
- 福島県：福島新教育研究協会（shinkyoken.com・新教研とは別法人の可能性、要個別確認）
- 教育開発出版株式会社（kyo-kai.co.jp）：単一県ではなく多県展開の模試教材サプライヤー。B2B窓口はhttps://www.kyo-kai.co.jp/contact/ で確認済みだが、「模試会社」というより教材出版社の性格が強く、位置づけを再検討してから追加するか判断
