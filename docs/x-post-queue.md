### ②公開ページ290本
```
気づいたらこのサイト、公開ページがsitemap提出ベースで290本を超えてた。全部ひとりで作ってる。
47都道府県ぶんの内申点・総合得点の計算方式が県ごとに全然違うので、正確に対応しようとすると
県別の解説とツールだけで数が増えていく。
```
ネタ源: GSC MCP `gsc_sitemaps`実測（2026-07-15時点でsubmitted=290）。

### ③47都道府県で内申点の中身が違う
```
「内申点」って言葉、47都道府県で計算方式が本当にバラバラ。東京都は実技4教科（音楽・美術・保健体育・
技術家庭）の評定が2倍で効く。埼玉県は中1〜中3の3学年ぶん全部が対象。神奈川県は「S値」という
独自の指標を使う。同じ「内申点」でも中身が全然違うので、全国共通の答えは存在しない。
```
ネタ源: src/lib/prefectures.ts実データで検証済み（東京=targetGrades:[3]・practicalMultiplier:2／
埼玉=targetGrades:[1,2,3]／神奈川=独自のS値方式で別エンジン実装）。
