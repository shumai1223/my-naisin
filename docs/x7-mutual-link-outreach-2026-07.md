# X-7 相互リンク打診 下書き集（第1弾・2026-07-23）

[[fable5-fullaccel-backlog-2026-07]] TIER X - X-7。対象は[[docs/backlink-gap-analysis-2026-07.md]]（X-31）で
発見したTier1（都道府県特化の個人・小規模塾運営者、自前の内申点計算ツールあり）。My Naishinは
全国47都道府県対応だが各県のニッチな一次情報（合格者データ・地域の入試傾向等）は持たないため、
「対等な運営者同士、守備範囲の違いを補い合う」という切り口で相互紹介を提案する。

**送信は一切していない（下書きのみ）**。対象4サイトはいずれもメールアドレスが非公開で、問い合わせフォーム
またはLINE公式アカウントのみが連絡手段（[[b2b-outreach-ready-2026-07]]のGmail下書き運用とは異なり、
フォーム入力は自動化できないため👤が手動でコピペ＋送信する運用になる）。件名はフォームに件名欄がある
場合のみ使用。

## 送信時の注意（👤向け）

- 4件ともメールでなく問い合わせフォーム/LINEが窓口のため、Gmail下書きのような自動設置はできない。
  👤が該当ページを開いて手動でコピペ＋送信する必要がある。
- 対象4（プラスジム）は唯一の直接競合（東京都・逆算ツール）のため、他3件より控えめな一方向提案に
  トーンを変えてある。送信するかどうかは👤の裁量に委ねる。
- 反応があった場合の返信は[[fable5-loop-protocol]]のV-0型運用に準じ、次回のGmail監視相当の
  仕組みがフォーム経由にはないため、👤が直接対応する前提。
