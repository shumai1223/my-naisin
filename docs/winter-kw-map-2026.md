## クラスタ別・現在の対象面と順位（GSC実測・90日）

### ① S値計算（神奈川県・倍率11.7xで最重要）
- 主対象面: `/kanagawa/s-value`
- `s値 計算`: 251imp / pos**1.97** / CTR56.6%（既に強い）
- `s値`: 379imp / pos**2.65** / CTR32.2%
- `s値計算ツール`: 55imp / pos**1.05**（独占）
- `s値とは`: **178imp / pos9.02 / CTR0%**（2026-07-20是正済み・21日ロック中〜08-10まで再編集不可）
- **判定**: 主要語は既に1-3位を確保。残る伸びしろは`s値とは`の効果測定待ち（08-10以降）と、`神奈川県公立高校 s値 一覧`(117imp/pos7.0)等のロングテール。

### ② 当日点計算（東京都・倍率7.3xで2番目）
- 主対象面: `/tokyo/total-score`
- `当日点計算 サイト`: 45imp / pos**1.51**（強い）だが同じクエリで`/`(home)が91imp/pos4.57・`/tokyo/naishin`が13imp/pos10.4と**カニバリ気味**（3ページが同一クエリで競合）
- `都立高校 当日点 計算方法`: 27imp / pos3.22（tokyo/total-score）+ 12imp/pos8.5（blog）
- `大阪府 公立高校 入試 内申 当日点 比率 2026`: **44imp / pos5.20**（osaka/naishin・伸びしろあり）
- **判定**: 東京は強いがhome/naishinとのカニバリ解消で更に伸ばせる余地。大阪の当日点比率クエリはpos5前後で改善余地あり。

### ③ 内申点計算（全国・倍率2.6x・head query）
- `内申点 計算`（home）: **9,412imp / pos3.72 / CTR7.9%**（全体の最大ボリューム・現状pos3-4で高止まり）
- 県別ロングテール（兵庫/大阪/神奈川）は軒並りpos4-5.5で安定
- **判定**: 母数最大のheadクエリがpos3-4滞留＝冬の絶対数インパクトが最も大きい。CTR7.9%は同順位帯の他クエリ（都立総合得点77.5%等）と比べ低く、改善余地は「順位」より「CTR」の可能性が高い（W-16定常監視の対象に追加検討）。
