# AA-1 追撃（フォローアップ）文面テンプレート

[[fable5-fullaccel-backlog-2026-07]] TIER AA-1。1通目無反応の相手への2通目は「催促」ではなく
「追加価値の提示」として書く（値を上げるための最小変更を持って再訪する形にする）。
**送信は👤の判断・実行のみ**。ここはテンプレの器のみで、実際に送る際は宛先ごとに固有名詞・
リンク・数値を差し込んで個別化すること（コピペ一斉送信はしない）。

候補の抽出は `npm run outreach:followup`（`data/outreach-ledger.json` × `src/lib/outreach-ledger.ts`）。

## 共通ルール

- 「ご確認いただけましたでしょうか」等の催促語彙は使わない。
- 必ず「1通目からの差分（新しく増えた価値）」を1つ以上入れる。差分が無い週は送らない。
- 件名は1通目と同一スレッドで返信する形（Re:）が基本。新規スレッドを起こす場合も1通目の
  件名を踏襲し、受け手が文脈を思い出せるようにする。
- 短く（3-5文）。長い追撃は読まれない。

## レーン1: b2b-saas（EdTech/模試会社/出版社等・API/MCP案内の追撃）

**追加価値の候補**（実在する差分のみ使う。無ければ送らない）:
- 内申点白書2026（`/report/2026`）を新規公開・都道府県別ダイジェストを順次追加中
- 開発者向けドキュメント（`/developers`）の更新・対応都道府県の増加

```
件名: Re: 内申点計算データAPI/MCPのご案内（My Naishin）

（会社名）ご担当者様

先日はご確認のお時間をいただき、ありがとうございました。
その後の状況共有までにご連絡いたしました。

先日ご案内した内容に加え、全国47都道府県の入試制度を比較した
「内申点白書2026」（https://my-naishin.com/report/2026）を新たに公開しました。
（会社名）様の（サービス名）とデータを組み合わせるイメージが湧きやすくなるかと思います。

もしご検討状況や、他にお伝えすべき情報がございましたらお知らせいただけますと幸いです。
引き続きどうぞよろしくお願いいたします。
```
