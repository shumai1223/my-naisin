# 競合被リンクギャップ分析（X-31・2026-07-23調査）

[[fable5-fullaccel-backlog-2026-07]] TIER X - X-31。目的はX-2/X-5/X-6/X-7/X-16の的中率を上げること。
「一度は同種のツールにリンクした/自分でツールを持っている実績がある」相手は初見の相手より心理的ハードルが
低い＝優先ターゲットという仮説の検証。**送信・打診は一切しない（調査のみ）**。

## 方法論と制約（正直に明記）

このloop環境にはAhrefs/SEMrush等の被リンク解析ツールも、GSC APIの「リンク」レポート（誰が自分に
リンクしているか）を返すMCPツールも無い（現行のgsc MCPはquery/totals/sitemaps/sitesのみ）。
そのため「my-naishin.comへの実際の被リンク元一覧」は取得不能。代わりに**WebSearch（Web検索）で
「内申点計算」「偏差値計算」関連の検索結果に登場する運営者・サイトを悉皆的に洗い出し、各サイトが
①自前の計算ツールを持つ運営者本人か（→相互リンクの対等な相手になり得る） ②ツールを持たず解説記事の
みか（→データ提供・寄稿先候補） ③直接競合で自サイトへのリンクは期待できないか、を分類する」という
代替手法を取った。実際に該当サイトがmy-naishin.comへリンクしているか/いないかの確認は個別ページの
目視・WebFetch確認をしたもののみ正確（未確認のものは「未確認」と明記）。

## 示唆（X-2/X-5/X-6/X-7/X-16への反映）

1. **X-7（独立教育系サイト運営者への相互リンク打診）の第一弾ターゲットはTier 1の11サイト**。
   「都道府県特化 vs 全国対応」という守備範囲の違いが対等な相互紹介の口実になる、既存想定より
   具体的で強い切り口。次にX-7に着手する周回はこの表からトーン別に3〜4件下書き化するとよい。
2. **X-2（地方紙悉皆）より先に、Tier 1の個人運営者の方が反応率が高い可能性がある**
   （地方紙は組織の意思決定が遅い一方、個人運営の塾サイトは即断即決しやすい）。X-2着手前に
   Tier1を消化する順序を推奨。
3. **X-27（メディア包括提案書）はbestjuku.com（塾選ジャーナル）を最優先候補にする**。
   寄稿10本（W-5）より一段大きい「継続データ提供パートナー」の器にふさわしい規模のメディア。
4. **X-16（模試会社再アプローチ）にdo-con.com（北海道学力コンクール系列の内申点計算ツール）を
   追加候補として合流させる**——模試会社が独自に内申点計算ツールまで持っている珍しいケース。
5. 今回の調査で「内申点計算サイトの被リンク元一覧」を直接取得する手段がこの環境には無いことを
   再確認した。将来的にGSCの「リンク」レポートAPI相当が使えるようになれば、本ドキュメントの
   Tier1候補が実際にmy-naishin.comへリンクしていないかの裏取りを優先すること（重複打診回避）。
