# 商談副操縦士 (meeting-copilot)

オンライン商談(Google Meet/Zoom)中に画面の隅に置く、リアルタイム回答補助ツール。

**設計思想: 自分が主・AIは横目の補助。** 知っている質問は「即答スクリプト」(遅延ゼロ)、想定外の質問だけAIボタン(2〜5秒)。

## 使い方

1. **商談の10分前**に `copilot-start.bat` をダブルクリック
   - サーバー起動+ブラウザが開く。AIのウォームアップに約40秒かかる(ヘッダーが「AI: 準備完了」になるまで待つ)
2. **イヤホンを外してスピーカー出音**にする(相手の声をマイクで拾って文字起こしするため)
3. 「🎙️ 文字起こし開始」を押してマイク許可
4. 商談中:
   - 来るとわかってる質問(価格/独占/実績/内製/無料枠…) → **即答スクリプト**をクリックで展開
   - 想定外の質問 → **⚡サジェスト**(Ctrl+Enterでも)。直近の会話ログがAIに渡り、3行以内のヒントが出る
   - 聞き取りミスは右下の手入力欄で追記できる

## 注意

- 音声・文字はどこにも保存しない(ブラウザを閉じたら消える)。ただし相手の発言をAI処理するので、**冒頭に「メモにAIツールを使っています」と一言断るのが推奨**(今のビジネスでは普通の断り)
- 文字起こしはChromeのWeb Speech API(要ネット接続・Chrome推奨)
- AI部分は claude CLI を stream-json 常駐モードで使用(APIキー不要・MCP非ロード)。モデルは既定 haiku(速度優先)。`COPILOT_MODEL=sonnet` で品質優先に変更可
- ファクト・交渉方針・相手情報は `prompt.md` に集約。**商談ごとにここを書き換える**
